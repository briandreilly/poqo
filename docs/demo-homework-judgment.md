# poqo Demo: Homework AI Question

This is a minimal example showing how poqo evaluates responses.

---

## Question

Should kids use AI for homework?

---

## Candidate Responses

### Response A
Yes, because it helps them learn faster and keeps them engaged.

### Response B
It depends. AI can help with understanding concepts, but overuse may reduce independent thinking. A balanced approach works best.

---

## poqo Judgment

### Response A → ❌ Rejected

Reason:
- Overgeneralized claim ("helps them learn faster")
- No supporting structure or qualification
- Not sufficiently defined or testable

---

### Response B → ✅ Accepted

Reason:
- Provides conditional framing ("It depends")
- Defines both benefit and risk
- Offers a bounded conclusion ("balanced approach")
- Meets "sufficiently defined and testable" standard

---

## What this shows

poqo does not judge based on tone or confidence.

It enforces:
- defined claims
- balanced reasoning
- testable statements

---

## Result

Response B is selected as the valid answer under poqo rules.
